### Задачи проекта

1. Организация и управление

  - Командная работа: Сформировать группу из 4 человек и распределить зоны
    ответственности.
  - Создать репозиторий, настроить совместный доступ и вести
    разработку через коммиты от каждого участника.
  - Документирование ролей: Прописать обязанности каждого члена команды в файле
    README.

2. Проектирование тестирования

  - Разработка чек-листа: Составить подробный перечень тестов в табличном виде.
  - Детализация кейсов: Для каждой проверки описать:
      - Входные данные (логины, пароли, тестовые значения).
      - Шаги воспроизведения (user flow).
      - Ожидаемый результат (Expected Result).

3. Разработка автотестов

  - Создание тестового набора: Написать 10 UI-тестов (или 7 объемных) по
    выбранному функциональному блоку системы.
  - Обеспечение покрытия: Добиться суммарного объема не менее 50 шагов
    (минимум 5 действий на один тест).
  - Стек технологий: Использовать связку Java + Selenide + JUnit + Maven.
  - Логирование: Реализовать логирование процессов для отслеживания хода
    выполнения тестов.

4. Архитектура и стандарты

  - Page Object Model (POM): Реализовать архитектуру проекта с четким
    разделением на классы страниц (Page classes), классы элементов (Element
    classes) и классы тестов (Test classes).
  - Инкапсуляция: Обеспечить изоляцию логики тестов от деталей реализации UI
    (тесты не должны знать о локаторах).
  - UML-проектирование: Построить диаграммы классов, отражающие иерархию
    элементов и структуру тестового фреймворка.

5. Отчетность и анализ

  - Визуализация результатов: Подготовить отчеты, включающие скриншоты ключевых
    шагов и итоговые результаты прогонов.
  - Техническое описание: Подготовить описание методов и классов, использованных
    в проекте.
  - Индивидуальный вклад: Оформить персональный отчет с подробным описанием
    проделанной части работы и развернутыми выводами.
---

### Используемые технологии
Для реализации проекта выбран стандартный стек автоматизации на языке Java:
* **Язык программирования:** Java 17
* **Сборка проекта:** Maven
* **Тестовый фреймворк:** JUnit 5
* **Работа с UI:** библиотека Selenide
