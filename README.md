# Автотесты для платформы Stepik.org

Данный репозиторий создан в рамках учебной практики (2 курс, направление ПМИ). Проект посвящен написанию автоматизированных тестов для образовательной платформы Stepik.

### Задачи проекта
В ходе работы планируется автоматизировать 5 функциональных блоков сайта. Согласно учебному заданию, проект разделен на два основных этапа:

1. **UI-тестирование** (воспроизведение реальных действий пользователя в браузере);
2. **Написание документации** (составление тест-кейсов для выбранных функций).

---

### Используемые технологии
Для реализации проекта выбран стандартный стек автоматизации на языке Java:
* **Язык программирования:** Java 17
* **Сборка проекта:** Maven
* **Тестовый фреймворк:** JUnit 5
* **Работа с UI:** библиотека Selenide


## Область тестирования
